# Tools

A curated list of tools on my machine. This includes both tools I actively use and tools I'm tracking for potential future installation.

## CSV Format

| Column | Description |
|--------|-------------|
| name | Tool name |
| category | Tool category |
| official_link | Official website or repository |
| install_method | Installation method (yay, pacman, npm, cargo, etc.) |
| description | Brief description |
| comment | Personal notes |
| install_date | Installation date (YYYY-MM-DD) |
